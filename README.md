# @salzpate React UI

React UI component library built with Tailwind CSS and Next.js 16

## Installation

```bash
npm install @salzpate/react-ui
# or
yarn add @salzpate/react-ui
```

## Setup

### Import Styles

Import the library styles in your main CSS file:

```css
/* app/globals.css or styles/globals.css */
@import '@salzpate/react-ui/styles';
```

Or in your Next.js layout/app:

```typescript
// app/layout.tsx
import '@salzpate/react-ui/styles';
```

**That's it!** With Tailwind CSS 4, all styles are included in the CSS bundle. No additional configuration needed.

## Getting Started (Development)

Run for development:

```bash
npm run storybook
# or
yarn storybook
```

Open [http://localhost:6006](http://localhost:6006) with your browser to see the result.
